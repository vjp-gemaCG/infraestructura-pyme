# Análisis de requisitos

## Cliente

PYME que necesita presencia web y sistema interno.

## Necesidades

- Servidor Linux Ubuntu Server 22.04
- Apache 2.4
- PHP 8
- MySQL 8
- Acceso SSH seguro
- Firewall UFW
- Monitorización
- Copias de seguridad automáticas

## Objetivos

Disponibilidad, seguridad y facilidad de mantenimiento.

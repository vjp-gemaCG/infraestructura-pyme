# Monitorización

## Herramienta elegida

Se utilizará **Netdata** para supervisar el estado del servidor en tiempo real.

## Parámetros a monitorizar

- Uso de CPU
- Uso de memoria RAM
- Espacio en disco
- Tráfico de red
- Procesos activos
- Estado del sistema

## Instalación prevista

```bash
bash <(curl -Ss https://my-netdata.io/kickstart.sh)
```

## Ventajas

- Interfaz web sencilla
- Datos en tiempo real
- Fácil instalación
